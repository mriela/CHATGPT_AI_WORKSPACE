# Poster Source Handoff

## 目的

このドキュメントは、MRIELA 側で先行作成したポスター制作準備資料を、SPALITUDA 側へ引き渡すためのメモです。

## MRIELA 側で作成済みの資料

| ファイル | 役割 |
|---|---|
| `assets/poster/source/poster_brief.md` | ポスター制作全体の仕様整理 |
| `assets/poster/source/poster_event_info_template.md` | 実イベント情報を記入するためのテンプレート |
| `assets/poster/source/poster_layout_text_draft.md` | ポスターの文字配置・構成案 |

## 現在の状態

- ポスター制作に必要なフォルダ構成は作成済み
- 実イベント情報の記入テンプレートは作成済み
- レイアウト案 A / B / C の初期整理は作成済み
- 画像素材そのものは未配置
- 実イベント情報は未確定

## SPALITUDA 側で次に行うこと

1. `assets/poster/source/poster_event_info_template.md` に実イベント情報を記入する
2. 必須素材がある場合は `assets/poster/references/` に配置する
3. レイアウト案 A / B / C のどれを使うか選ぶ
4. 初回デザイン案を `assets/poster/drafts/` に作成する
5. 書き出し画像を `assets/poster/exports/` に保存する
6. Unity / VRChat 掲示用に調整した素材を `assets/poster/unity-ready/` に保存する

## 注意事項

- 未確定情報は `TBD` のまま残す
- 出典不明素材は成果物へ直接使用しない
- Unity掲示用は遠目の可読性を優先する
- SNS用とUnity用は同一デザインから派生できる構成が望ましい

## SPALITUDA への依頼文案

SPALITUDA 側では、まず以下を確認してください。

```text
MRIELA 側で作成されたポスター準備資料を確認してください。

確認対象：
- assets/poster/source/poster_brief.md
- assets/poster/source/poster_event_info_template.md
- assets/poster/source/poster_layout_text_draft.md
- handoff/to-SPALITUDA/poster_source_handoff.md

目的：
実イベント用ポスター制作に向けて、未確定情報・必要素材・初回レイアウト方針を確認し、次に必要な作業を整理してください。
```
