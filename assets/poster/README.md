# Poster Assets

Purpose:
- Manage poster-related materials for the VRChat音楽イベント on MRIELA.
- Keep draft assets, source notes, and final handoff files organized before SPALITUDA integrates them with the Unity venue.

## Folder Purpose
- Store materials used to create event posters on MRIELA.
- Keep poster drafts and export files separate from general project documents.
- Make it easy for SPALITUDA to find the latest poster assets after pulling from GitHub.

## Planned Assets
- Poster draft images
- Final poster exports
- Background images or references
- Event logo and organizer logo files
- QR codes or URL graphics
- Font and color reference notes
- In-world display variants for Unity venue signage

## File Naming Rules
- Use lowercase English filenames where possible.
- Use hyphens or underscores instead of spaces.
- Include the event or asset type in the filename.
- Include a version number for drafts.
- Use clear suffixes such as `_draft`, `_final`, `_unity`, or `_qr`.

Examples:
- `vrchat-music-event_poster_draft_v01.png`
- `vrchat-music-event_poster_final.png`
- `vrchat-music-event_unity_sign_v01.png`
- `vrchat-music-event_qr.png`

## Final Deliverables for SPALITUDA
- Final poster image for announcement use
- Unity-ready poster or sign texture
- Any required logo or QR code assets
- Notes about colors, fonts, aspect ratio, and intended placement
- Source files if they are needed for later edits

## Unity Venue Worldview Notes
- Match poster colors with the Unity venue lighting and atmosphere.
- Keep text readable at VRChat viewing distance.
- Confirm aspect ratio before importing poster textures into Unity.
- Avoid overly large texture files that may affect world performance.
- Use the same visual motifs as the selected venue direction, such as stars, cafe warmth, or neon club elements.
