# Poster Brief

## 目的

このドキュメントは、実イベント用ポスター制作に必要な仕様・素材・未確定事項を整理するためのブリーフです。

MRIELA は、PC-A / SPALITUDA が重作業中でも先行してポスター関連の準備を進める補助作業機として運用します。

## 想定成果物

- VRChatイベント告知用ポスター
- SNS告知用画像
- Unity / VRChatワールド内掲示用画像
- 必要に応じた縦横比違いの派生版

## 現時点の確定事項

- GitHubリポジトリ `mriela/CHATGPT_AI_WORKSPACE` を共通正本とする
- ポスター関連素材は `assets/poster/` 配下で管理する
- 制作元ファイルは `assets/poster/source/` に置く
- 書き出し画像は `assets/poster/exports/` に置く
- Unity向け調整済み素材は `assets/poster/unity-ready/` に置く

## 未確定事項

以下は、実制作前に SPALITUDA またはユーザー側で確定する必要があります。

| 項目 | 状態 | メモ |
|---|---|---|
| イベント名 | 未確定 | 正式名称が必要 |
| 開催日時 | 未確定 | 日付・開始時刻・終了時刻 |
| 会場 / ワールド名 | 未確定 | VRChatワールド名など |
| 主催者名 | 未確定 | 表記ゆれ注意 |
| 出演者 / 参加者 | 未確定 | 必要なら一覧化 |
| テーマ / 雰囲気 | 未確定 | 例：近未来、幻想的、音楽イベント等 |
| 必須ロゴ | 未確定 | 使用許諾も要確認 |
| 必須テキスト | 未確定 | 告知文・注意事項など |
| 出力サイズ | 未確定 | SNS用 / Unity用で分ける可能性あり |
| 納品形式 | 未確定 | PNG / JPG / PSD / Clip / SVG等 |

## 推奨する初期出力サイズ

| 用途 | 推奨サイズ |
|---|---|
| SNS告知用 正方形 | 1080 x 1080 |
| SNS告知用 縦長 | 1080 x 1350 |
| X / Twitter横長 | 1600 x 900 |
| Unity掲示用 | 2048 x 2048 または 2048 x 1024 |

## 制作時の注意

- 文字情報は後から差し替えやすい構成にする
- Unity掲示用は遠目でも読めるよう、文字サイズとコントラストを優先する
- SNS用とUnity用は、同じデザインから派生できる構成にする
- 未確定情報は仮置きせず、`TBD` と明記する
- 素材の出典・使用許諾が不明なものは `references/` に置くだけにし、成果物へ直接使用しない

## 次に必要な入力

ユーザーまたは SPALITUDA から、最低限以下を受け取る必要があります。

1. イベント名
2. 開催日時
3. 会場 / ワールド名
4. 告知文
5. 画像の雰囲気
6. 出力サイズ
7. 使用必須素材の有無

## MRIELA の担当範囲

MRIELA は以下を先行して実施できます。

- ブリーフ整理
- 必要情報チェックリスト作成
- デザイン案の構成整理
- 素材フォルダ整理
- 仮レイアウト案のテキスト化
- SPALITUDA への引き渡し文書作成

ただし、Unity上での最終確認や重い画像生成・編集は、必要に応じて SPALITUDA 側で実施します。
