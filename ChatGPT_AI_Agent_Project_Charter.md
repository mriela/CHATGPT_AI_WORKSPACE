# ChatGPT中心のAI Agent計画

## プロジェクト目的

ChatGPTはAI Agentを操作できるか？

## 成功条件

```text
あなた
↓
ChatGPT
↓
AI Agent
↓
System

(Unity / VSCode / Browser / Discord 等)
```

## 重要な判断基準

以下は目的ではなく手段である。

- ChatGPT → Unity
- ChatGPT → VSCode
- ChatGPT → Chrome
- ChatGPT → Discord

評価すべき対象は、

```text
ChatGPT
↓
AI Agent
```

が成立しているかどうかである。

## 今後の評価方針

新しい作業や技術検証を行う際は、

「その作業は ChatGPT → AI Agent の実証に近づくか？」

を基準に優先順位を判断する。
