# MRIELA Role

MRIELA is the lightweight preparation workstation.

## Purpose
- Prepare work while SPALITUDA is busy
- Create documents, posters, assets, notes, and handoff materials
- Push prepared work to GitHub so SPALITUDA can resume later

## Rule
- MRIELA does not perform final heavy Unity work
- SPALITUDA is the main machine for Unity, VRChat SDK, builds, and heavy GPU tasks

## Handoff
- MRIELA -> SPALITUDA: handoff/to-SPALITUDA/
- SPALITUDA -> MRIELA: handoff/from-SPALITUDA/
